:active

O :activeseletor é usado para selecionar e estilizar o link ativo. Um link se torna ativo quando você clica nele. O seletor :active pode ser usado em todos os elementos, não apenas em links. :active DEVE vir depois de :hover (se presente) na definição CSS para ser eficaz!

::after

O ::after insere algo após o conteúdo de cada elemento(s) selecionado(s) Use a propriedade "content" para especificar o conteúdo a ser inserido.

eu sou nerd

eu sou rico

::before

O ::before insere algo antes do conteúdo de cada elemento(s) selecionado(s). use a propriedade "content" para especificar o conteúdo a ser inserido.

I love you :)

I hate you >:(

:checked

O :checked corresponde a cada elemento "input" verificado (apenas para botões de opção e caixas de seleção) e elemento "option".






:default

O :default seleciona o elemento verificado (checked) de formulário em um grupo de elementos relacionados. O seletor :default só pode ser usado nos elementos "button", "input type="checkbox", "input type="radio" e "option"
