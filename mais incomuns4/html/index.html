::marker

:nth-child()

O seletor corresponde a cada elemento que é o enésimo filho de seu pai. :nth-child(n) n pode ser um número, uma palavra-chave (ímpar ou par) ou uma fórmula (como an + b ).

:nth-last-of-type(n)

O seletor corresponde a cada elemento que é o enésimo filho, de um tipo específico, de seu pai, contando a partir do último filho. :nth-last-of-type(n) n pode ser um número, uma palavra-chave ou uma fórmula. Ímpar e par são palavras-chave que podem ser usadas para corresponder a elementos filhos cujo índice é ímpar ou par (o índice do primeiro filho é 1).
