:first-child

O :first-child é usado para selecionar o seletor especificado somente se ele for o primeiro filho de seu pai. É preciso que o elemento filho esteja logo após o elemento pai.

:first-of-type

O :first-of-typeseletor corresponde a cada elemento que é o primeiro filho, de um tipo específico, de seu pai. O elemento filho não precisa estar logo em seguida do elemento pai.

:focus

O :focus é usado para selecionar o elemento que está em foco. Dica: O seletor :focus é permitido em elementos que aceitam eventos de teclado ou outras entradas do usuário.




:hover

O :hoverseletor é usado para selecionar elementos quando você passa o mouse sobre eles. :hover DEVE vir depois de :link e :visited (se estiverem presentes) na definição CSS, para ser eficaz!

:in-range

O :in-range seleciona todos os elementos com um valor que esteja dentro de um intervalo especificado. O seletor :in-range funciona apenas para elementos de entrada com atributos min e/ou max!

:invalid

O :invalid seleciona elementos do formulário com um valor que não é validado de acordo com as configurações do elemento. O seletor :invalid funciona apenas para elementos de formulário com limitações, como elementos de entrada com atributos mínimo e máximo, campos de e-mail sem e-mail legal ou campos numéricos sem valor numérico, etc.

:lang()

O :lang() é usado para selecionar elementos com um atributo lang com a língua especificada. Observação: o valor do atributo lang geralmente é um código de idioma de duas letras, como lang="fr" (para francês), ou dois códigos de idioma combinados, como lang="fr-ca" (para francês canadense).
